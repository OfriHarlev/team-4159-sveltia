<h1>Parallax Scrolling</h1>
<p>Adversary utilizes an effect known as &#8220;parallax scrolling&#8221; (different elements on the page scroll at different speeds) on the header images to present your content in a dynamic and interactive way.</p>
<p>&nbsp;</p>
<h1>The Index Collection</h1>
<p>Build your presentation-style layout by adding pages to the Index Collection. The&nbsp;Thumbnail&nbsp;Image for each page will be its &#8220;header&#8221; image. The Thumbnail Image, Page Title, and Description are set in Page Settings.</p>
<p>&nbsp;</p>
<h1>Other notable features</h1>
</p>
<h3>Turn off parallax scrolling</h3>
<p><span style="font-size: 14px; font-weight: normal; line-height: 1.68em;">You can turn off the parallax scrolling effect if you just want a simple single-page layout.&nbsp;</span></p>
</p>
<h3>First Index Image Fullscreen</h3>
<p>You can choose to have the first image in the Index collection be fullscreen or match the height of the other images.&nbsp;</p>
<h3>Hide Page Title</h3>
<p><span>You can choose to hide the page/collection titles.&nbsp;</span></p>
</p>
</p>
<h3>Hide Parallax Nav</h3>
<p>You can hide the index nav.&nbsp;</p>
<h3>Index Image Height</h3>
<p>You can set the height of all the index images to be full-screen, two-thirds or half.</p>
<h3>Title &amp; Description Position</h3>
<p>You can choose to place the Title &amp; Description on top of or under the main image.</p>
</p>
<h3>Footer Background Color</h3>
<p>Set the footer background color. The color of the text in the footer will be automatically set to white or black depending on the lightness of the color you chose to ensure it is legible.&nbsp;</p>
<p>&nbsp;</p>
